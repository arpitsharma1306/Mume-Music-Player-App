# Assets Folder

This folder contains the app's static assets.

## Required Files

You need to create the following image files:

1. **icon.png** (1024x1024) - App icon
2. **splash.png** (1284x2778) - Splash screen image
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon foreground
4. **favicon.png** (48x48) - Web favicon

## Creating Placeholder Icons

You can use any image editing tool to create these. For quick testing, you can:

1. Use a solid color background (#1A1A2E - dark purple)
2. Add the text "MUME" in the center (#E91E63 - pink)
3. Export at the required sizes

## Color Scheme

- Primary: #E91E63 (Pink)
- Background: #1A1A2E (Dark Purple)
- Text: #FFFFFF (White)
